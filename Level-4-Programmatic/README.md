# Level 4: Programmatic Usage - 编程式使用

## 📚 本级别学习目标

掌握 Claude Code 的编程式集成方法,包括无头模式、SDK 使用和 CI/CD 集成。

## 📖 学习内容

- **4.1 Headless Mode** - --output-format json、Stream processing、Exit codes
- **4.2 Python SDK** - ClaudeCodeOptions、ClaudeCodeClient 使用、Async streaming
- **4.3 TypeScript SDK** - query() 函数使用、Message types、SDK MCP servers
- **4.4 GitHub Actions** - @clibot/codebot-cli 配置、Workflow configuration

> 注: 本级别内容主要基于官方文档，适合需要自动化集成的开发者

## 🎯 学习检查点

完成本级别学习后,你应该能够:

- ✅ 在脚本中以无头模式使用 Claude Code
- ✅ 使用 Python/TypeScript SDK 集成 Claude Code
- ✅ 在 GitHub Actions 中自动化 Claude Code 工作流
- ✅ 处理 JSON 输出和流式响应
