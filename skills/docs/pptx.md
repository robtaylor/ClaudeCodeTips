# 📙 PPTX - PowerPoint 演示文稿处理技能

## 概述

PPTX skill 提供全面的 PowerPoint 演示文稿创建、编辑和分析能力，支持幻灯片布局、动画、演讲者备注和注释。

**适用场景**：
- 创建新的演示文稿
- 修改现有幻灯片内容
- 提取演示文稿文本和备注
- 批量生成个性化演示文稿
- 分析演示文稿结构

---

## 🎯 核心功能

### 1. 演示文稿创建
使用 html2pptx 工作流从 HTML 转换为 PowerPoint。

**支持特性**：
- ✅ 自定义主题和配色
- ✅ 精确的布局控制
- ✅ 图片和媒体嵌入
- ✅ 表格和图表
- ✅ 动画和过渡

### 2. 内容提取
从现有演示文稿中提取内容。

**支持特性**：
- ✅ 文本提取（Markdown格式）
- ✅ 演讲者备注提取
- ✅ 注释读取
- ✅ 媒体文件导出

### 3. 演示文稿编辑
修改现有演示文稿的内容和格式。

**支持特性**：
- ✅ 文本替换
- ✅ 幻灯片添加/删除
- ✅ 布局修改
- ✅ 主题应用

## 🔧 使用方法

### 场景 1: 创建新演示文稿

**示例需求**：
```
创建一份产品发布演示文稿，包括5个幻灯片：
1. 标题页
2. 产品特性
3. 市场分析
4. 竞争优势
```

**处理流程**：
1. 分析内容，选择合适的设计风格
2. 使用 HTML 创建幻灯片内容
3. 转换为 PowerPoint
4. 应用主题和格式

### 场景 3: 提取演讲者备注

**示例需求**：
```
从这份演示文稿中提取所有演讲者备注
```

**使用 markitdown**：
```bash
python -m markitdown presentation.pptx > content.md
```

**或使用 OOXML**：
```bash
# 解包
python ooxml/scripts/unpack.py presentation.pptx unpacked/

# 读取备注
cat unpacked/ppt/notesSlides/notesSlide1.xml
```

1. 解包演示文稿

## 💡 设计原则

### 内容驱动设计

在创建演示文稿前，需要考虑：

1. **主题适配** - 内容是关于什么的？技术？商业？教育？
2. **品牌识别** - 是否有企业品牌色？
3. **配色选择** - 颜色要反映内容调性
4. **版式布局** - 专业、现代、创意？

### 颜色方案示例

| 主题 | 主色 | 辅色 | 用途 |
|------|------|------|------|
| 科技 | 蓝色 | 银灰 | 技术产品演示 |
| 金融 | 深蓝 | 金色 | 财务报告 |
| 教育 | 绿色 | 橙色 | 培训课程 |
| 创意 | 紫色 | 粉色 | 设计提案 |

## ⚠️ 注意事项

### 最佳实践

**✅ 应该做的**：
- 创建前声明设计思路
- 选择内容匹配的配色
- 保持版式一致性
- 使用高质量图片
- 简洁的幻灯片内容

**❌ 不应该做的**：
- 使用默认模板不加修改
- 幻灯片内容过度拥挤
- 忽略品牌识别
- 动画过度使用
