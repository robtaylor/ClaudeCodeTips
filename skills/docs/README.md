# Claude Code Skills 使用文档

本目录包含了各个 Claude Code Skills 的详细使用指南。

## 📚 可用文档

### 🔍 信息验证工具

#### [Source Tracing - 信息溯源技能](source-tracing.md) ⭐ 重点推荐

从高熵的二手内容中回溯到低熵、权威、原始的信息源。

**核心价值**:
> 追溯信息源，一层一层往下挖，去读、去看最原始的材料，就已经领先了至少 99% 的人。

**主要功能**:
- 🎯 六步溯源流程（去包装 → 源头定位 → 原文解读 → 语境还原 → 多源对照 → 结构存档）
- 📊 可信度评分体系
- 📝 结构化验证报告生成
- 🔗 权威信息源清单

**适用场景**:
- 政策解读验证
- 突发新闻核实
- 行业事件溯源
- 学术研究验证

## 🚀 如何使用

### 方法 1: 直接调用
在 Claude Code 中直接描述您的需求即可。

### 方法 2: 查看技术文档
每个 skill 的完整技术规范位于：
```
.claude/skills/<skill-name>/SKILL.md
```

### 方法 3: 阅读使用指南
本目录下的 Markdown 文件提供了详细的使用指南和实战案例。

## 📝 文档贡献

如果您想为其他 Skills 添加使用文档，请遵循以下规范：

### 文档结构
```markdown
# Skill 名称

## 概述
简要介绍 skill 的用途和核心价值

## 核心功能
列出主要功能点

## 使用场景
提供具体的使用场景示例

## 使用方法
详细的操作步骤

## 实战案例
真实的使用案例和结果

## 进阶技巧
高级用法和最佳实践

## 注意事项
局限性和使用建议
```

### 命名规范
- 文件名使用小写字母和连字符
- 例如：`source-tracing.md`、`pdf-processing.md`

### 内容要求
- 使用中文编写，专业术语保留英文
- 提供真实案例和截图（如适用）
- 包含代码示例和配置示例
- 突出实用性和可操作性
